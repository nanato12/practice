## ビット演算

これも他の言語と変わらない
一応
| 演算子 | 説明 | 使用例 |
| :--- | :--- | :--- |
| \| | 論理和 | 0x10|0x01 |
| & | 論理積 | 0x1&0xf |
| ^ | 否定 | ^0x3 |
| ^ | 排他的論理和 | 0xc^0x3 |
| &^ | 論理積の否定 | 0xc&^0x3 |
| >> | 右に算術シフト | 0x4<<4 |
| << | 左に算術シフト | 0x4>>1 |
